Scientific and Graphing Calculator
A javafx calculator application that can be used as a scientific calculater as well as allows users to input mathematical expressions and visualize their corresponding graphs. This tool supports a variety of expressions, including basic arithmetic operations and trigonometric functions like sin, cos, tan, csc, cot, and sec.

Description
A very basic ui for simple calculator and also provides an intuitive interface to analyze mathematical functions visually. Users can input expressions, view their graphical representations, and interact with the graph by moving or zooming in/out. The sidebar also displays a history of expressions entered.

Features:
Expression Input: Type and evaluate expressions with trigonometric functions and arithmetic operators.

Supported Functions: Graph basic expressions like y=x^2 or trigonometric expressions like cos(x^3) + 10sin(y) and log functions
